### 확정 파라미터

| 파라미터 | 값 | 설명 |
|---|---|---|
| rsiPeriod | 14 | RSI 계산 기간 |
| rsiOversold | **35** | 매수 진입 기준 (최적값) |
| rsiOverbought | 70 | 매도 청산 기준 |
| macdShortPeriod | 12 | MACD 단기 EMA |
| macdLongPeriod | 26 | MACD 장기 EMA |
| macdSignalPeriod | 9 | MACD 시그널 EMA |
| atrPeriod | 14 | ATR 계산 기간 (변동성 필터) |

### 진입/청산 조건

**매수 조건 (3가지 동시 충족)**
1. RSI < 35 — 과매도 구간 진입
2. MACD > 시그널선 — 상승 모멘텀 확인
3. ATR > SMA(ATR, 28) — 평균 이상 변동성 구간 (저변동 구간 제외)

**매도 조건 (하나라도 충족 시 청산)**
- RSI > 50 — 과매도 해소, 평균 회귀 완료
- 고정 손절 -10% — 급락 안전망 (1년간 미발동, 퍼포먼스 영향 없음)

## 최적화 과정

### 1단계 — 기준선 측정 (5분봉, RSI30)

| 항목 | 값 |
|---|---|
| 백테스팅 id | 5 |
| 수익률 | -3.45% |
| 거래 횟수 | 85 |
| 최대낙폭 | 9.21% |
| 손익비 | 1.306 |

**문제**: 85거래 × 0.1% 수수료 ≈ 8.5% 수수료 부담으로 수익 잠식.
전략 패턴 자체(승률 54%, PF 1.306)는 양호하나 스캘핑 특성상 수수료에 잠식됨.

|---|---|

| 수익률 | -0.93% |
| 거래 횟수 | 17 |
| 최대낙폭 | 3.79% |

**효과**: 거래 85 → 17로 감소, 손실 -3.45% → -0.93% 개선.
**한계**: 여전히 마이너스. 수수료 절감 효과는 있으나 진입 조건이 과도하게 제한됨.

## 유의사항

- 검증 기간(1년, 33거래)은 통계적으로 충분하지 않을 수 있음. 2~3년 이상 데이터로 추가 검증 권장.
- 현재 손절 조건 없음 (RSI > 50 청산만 존재). 급락 시 손실 확대 위험.
- 업비트 수수료 0.05% 기준 적용. 수수료 변경 시 재검증 필요.
- 과거 성과가 미래 수익을 보장하지 않음.
